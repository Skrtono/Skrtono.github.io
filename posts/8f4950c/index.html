利用Geyser 让Java玩家和基岩玩家一同游玩
发表于 2020-03-11| 更新于 2020-03-12|Minecraft
字数总计:818|阅读时长: 3 分钟
| 阅读量:|评论数:

由于迟迟无法开学，舍友们找我说要一起玩MC，但分别用的是手机或电脑。
我之前看到过某些文章，这是可行的。便开始折腾了。

这两天，经过多次测试，总数是敲定一个方案了。
官服 + Geyser

Geyser是最近一段时间出现的，虽然还在测试阶段，但我觉得他是众多方案当中最为方便的一个

先介绍一下Geyser
官网 Github 下载
由一个团队构建而成，成员目前只有3人，分别是：RednedEpic,SupremeMortal,Tim203
Geyser能将基岩版玩家的一举一动 转换模拟为Java版本的操作。
但目前还有限制，那就是你只能选择1.15.2来作为服务器，Bug不是特别影响游戏体验
master分支无法合成，强烈建议使用inventory测试分支
当然你也可以使用viaversion来进行转换，来再其他版本上游玩。
目前分为两个版本，一个是代理服，一个是作为插件来运行。

本人是选择代理服的。
我这里就不多说开服过程了，具体自行百度。
直接讲Geyser的运行配置

首先，你必须先要开好一个1.15.2的服务器
然后运行再来运行Geyser
对于Geyser 512M就够了

Windows 用户
1、建立一个bat运行脚本
java -Xms512M -jar Geyser.jar
2、命令行到当前文件夹
java -Xms512M -jar Geyser.jar

Liunx 用户
1、建立一个sh
#!/bin/sh BINDIR=$(dirname “$(readlink -fn “$0”)”) cd “$BINDIR” java -jar Geyser.jar
2、命令行到当前文件夹
java -jar Geyser.jar

开启之后，Geyser会生成它的配置文件。
